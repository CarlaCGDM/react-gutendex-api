(inner) validarPassword(password) → {boolean}

Comprueba que la contraseña que el usuario intenta crear para su cuenta cumpla con los requisitos de seguridad básicos mediante una expresión regular (la constraseña debe tener entre 8 y 16 caracteres alfanuméricos y al menos una mayúscula, una minúscula, un número y un caracter especial).
